# 📍 Google Maps Recorder – Chrome Extension

A lightweight Chrome extension that extracts business listings from Google Maps search results and exports them as CSV or JSON.

Built for fast data collection and research workflows.

---

## 🚀 Features

- Extract business listings from Google Maps search results
- Export results as CSV or JSON
- Live preview inside floating widget

---

## 🧠 How It Works

1. Open Google Maps
2. Search for a keyword (e.g. “Nasi Lemak Malaysia)
3. Scroll down a few times to load more results
4. Click **Extract Feed**
5. Download data as CSV or JSON

---

## 📦 Installation (From GitHub)

### 1. Download the Extension

Go to the GitHub repository:

👉 https://github.com/your-username/google-maps-recorder

Click the green **Code** button, then select:

**Download ZIP**

---

### 2. Extract the Files

- Locate the downloaded ZIP file (usually in your Downloads folder)
- Right-click → **Extract All**
- Open the extracted folder

---

### 3. Open Chrome Extensions Page

Open Google Chrome and go to setting & look for "Extensions" > "Manage Extensions". You should be redirected to chrome://extensions/

---

### 4. Enable Developer Mode

In the top-right corner, turn ON:

**Developer mode**

---

### 5. Load the Extension

Click the button:

**Load unpacked**

Then select the extracted folder from Step 2.

---

### 6. Done 🎉

Go to Google Maps:

https://www.google.com/maps

You will see the **Google Map Recorder widget** appear automatically on top right of the page.

---

## 🖥️ Widget UI

The floating widget includes:

- Extract results from "Search Google Maps"
- Download CSV / JSON
- Live preview
